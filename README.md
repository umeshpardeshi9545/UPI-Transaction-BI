# UPI-Transaction-BI

# Description

This repository contains an interactive Power BI Dashboard built to analyze UPI (Unified Payments Interface) transactions across customers, banks, cities, payment methods, devices, and transaction statuses.

The dashboard provides both transaction-level insights and high-level summary metrics, enabling quick business decisions and deeper analytical exploration.

Success vs Failed transaction analysis

It is designed for real-time monitoring, performance analysis, and decision support.
